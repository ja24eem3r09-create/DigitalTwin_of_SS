# Digital Twin of Substation Transformer 🔌

## Overview
A Digital Twin of a Substation Transformer developed using 
Unity 3D and Machine Learning, featuring real-time monitoring,
fault detection, temperature prediction, and load forecasting.

## Tech Stack
- 🎮 Unity 3D - 3D Visualization & Simulation
- 💻 C# - Unity Scripting
- 🐍 Python - Machine Learning Models
- 📡 RTU/IoT - Real-time Data Integration

## Features
- ✅ Real-time Monitoring
- ✅ Fault Detection
- ✅ Temperature Prediction
- ✅ Load Forecasting

## Project Structure
- /Scenes - Unity Scene files
- /Scripts - C# Unity scripts
- /PythonProject10 - ML models & dashboard
